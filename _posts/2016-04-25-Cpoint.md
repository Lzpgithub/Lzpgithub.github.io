在学习指针之前，必须弄清楚“内存单元的地址”和“内存单元的内容”的区别。

指针变量：存储地址的变量，编译系统一般分配存储4个字节的存储单元。

注意：（1）不能用一个整数给指针变量赋值
     （2）定义指针变量要指明类型

### 初始化方式的总结： 

1

```cpp
int a=5;
int *p1;
p1=&a;
int *p2=&a;
```

2 int *p=NULL  赋值为空指针

3 字符串指针、数组赋值

```cpp
char *str="hello";
char *shrs[]="hello";
char *p[]={"aaa","bbb"};   //字符串指针数组赋值  不可以:int *p[]={1,2};
```

4 数组的第一个元素的地址赋值给指针

```cpp
int a[3]={1,2,3};
int *p;
p=&a[0];  //p=a
```

### 相关程序1

```cpp
#include <stdio.h>
#include <stdlib.h>
#include <iostream>

using namespace std;

int main(){
	int *i;
	int a=5;
	i=&a;
	cout<<i<<" "<<i+1<<endl;    //输出相邻的地址

	char *c;
	char b='c';
	c=&b;
	printf("%0X %0X\n",c,c+1);  //输出相邻地址

	int d[2]={1,2};
	cout<<d<<" "<<d+1<<endl;    //输出相邻地址

	int *s=d; 
	cout<<sizeof(s)<<" "<<sizeof(d)<<endl;   //输出指针和数组的大小
	cout<<&d<<" "<<&d+1<<endl;  //输出相邻地址

	system("pause");
}
```
